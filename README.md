# Single Agent

A single agent implementation for autonomous task execution and decision-making.

## Overview

This project contains implementations of a single agent that can perform various tasks using AI-driven logic and tool integration.

## Files

- `Untitled.ipynb` - Single agent notebook
- `Untitled1.ipynb` - Additional agent implementation

## Requirements

- Python 3.8+
- Jupyter Notebook
- Dependencies listed in requirements (see notebooks for details)

## Usage

Open the notebooks in Jupyter and run the cells to execute the single agent implementations.

```bash
jupyter notebook
```
